# Fee Management System - Professional "One-Click" Setup

This project is configured as a professional software package. Follow these steps to prepare the setup for your client:

## 1. Preparation (Zipping)
- Go to your folder: `fee-management-system`
- **REQUIRED**: Delete these folders to keep the package small:
  - `.next`
  - `node_modules`
- Zip the entire `fee-management-system` folder. Name it: `Fee_Management_Professional_Setup.zip`.

## 2. Instructions for the Client
Send the zip file to your client with these **3 simple steps**:

### Step 1: Extract All
**IMPORTANT**: Right-click the zip file and select **"Extract All..."**.
- Move the extracted folder to a permanent place (like Desktop or Documents).

### Step 2: Professional Installation
Double-click **`INSTALL_ERP.bat`**.
- It will automatically ask for **Administrator Permission** (Click "Yes").
- It will setup everything: Node.js, dependencies, and a **Desktop Shortcut**.
- *Wait for the "Installation Successful" message.*

### Step 3: Start from Desktop
You don't need to open folders anymore. Just double-click the **"Fee Management System"** icon on your Desktop.
- The system starts in the background (no black window).
- Your browser will open the Dashboard automatically.
